# Cyrilex 

Cyrilex is a visual regex tester.

Try the demo here: [https://extendsclass.com/regex-tester.html](https://extendsclass.com/regex-tester.html)

## Alguns exemplos de uso:

```
/(XP[a-z]th)|(JSONPath)/g

Online XPath tester


/(XPth)|(Ribafs)/g

Ribafs


/(Ribafs)/g


/[a-z]/g

r


/[a-z]-[1-3]/g

adfdfd-3
```
